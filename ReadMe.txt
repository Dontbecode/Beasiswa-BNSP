# Proyek Website BNSP-BEASISWA

## Struktur Folder

- **Components**: Folder ini berisi komponen untuk tampilan admin dan user.
  - **Admin**: Komponen untuk tampilan admin.
  - **User**: Komponen untuk tampilan user.

- **Controllers**: Folder ini berisi logic atau pemrosesan data.
  - **Admin**: Logic atau pemrosesan data untuk admin.
  - **User**: Logic atau pemrosesan data untuk user.

- **Doc**: Tempat berkas yang terupload.

- **IMG**: Folder untuk asset gambar yang digunakan di website.

- **Public**: Penyimpanan file CSS dan JS.

- **Connect.php**: File untuk koneksi ke database.

- **DashboardAdmin.php**: Halaman utama untuk admin.

- **Index.php**: Halaman utama yang dijalankan oleh website.
